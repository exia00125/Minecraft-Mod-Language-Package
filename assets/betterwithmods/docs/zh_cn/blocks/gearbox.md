# 变速箱

![变速箱](block:betterwithmods:gearbox@0)

变速箱用于传递机械动力。
动能必须输入到这个面，并可以输出到任何其他面。

![输入面](betterwithmods:textures/blocks/gearbox_input.png)

可以通过Shift+右键点击来使其旋转，以使其更容易使用。
